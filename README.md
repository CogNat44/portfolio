A selection of web projects built for Cognition Controls.

## index.html — Website Home Page Revamp

Revamped the Cognition Controls home page with a focus on trust and conversion.

## cognition-case-studies.html — Case Study Template & Content

Developed a reusable case study format and wrote five case studies covering relevant customer types and problem areas.

- Built a single-page template with consistent structure: hero image, stat cards, challenge block, body sections, trusted brands strip, and CTA
- Wrote case studies for: retail multi-site rollout (NAPA), QSR refrigeration & HVAC (Burger King), municipal public buildings, automotive dealership, and predictive freezer monitoring
- Each case study focuses on a distinct problem area — energy savings, onsite behaviour change, remote management, equipment intelligence, and predictive alerting
